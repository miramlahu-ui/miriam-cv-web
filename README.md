# Miriam web visual

Proyecto listo para volver a subir a GitHub y publicar en Vercel.

## Qué trae
- foto real en `public/assets/miriam-photo.jpg`
- miniaturas visibles de certificados y CV
- PDFs reales dentro de `public/assets/`
- diseño moderno con bloques visuales

## Para subirlo
1. Descomprime esta carpeta.
2. En tu repositorio de GitHub, elimina los archivos viejos si hace falta.
3. Sube todo el contenido de esta carpeta.
4. Vercel volverá a publicar la web automáticamente.

## Archivos importantes
- `src/App.jsx`
- `src/styles.css`
- `public/assets/`
