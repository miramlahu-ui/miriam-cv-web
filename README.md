# Miriam CV Digital

Proyecto listo para publicar en Vercel con React + Vite.

## Cómo verlo en tu computadora

1. Instala Node.js.
2. Abre esta carpeta en tu terminal.
3. Ejecuta:

```bash
npm install
npm run dev
```

## Cómo publicarlo en Vercel

1. Sube esta carpeta a GitHub.
2. Entra a Vercel.
3. Pulsa **New Project**.
4. Elige tu repositorio.
5. Pulsa **Deploy**.

## Archivo principal para editar

`src/App.jsx`

Ahí puedes cambiar experiencia, formación, contacto y cursos.
